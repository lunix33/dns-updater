Usage

With a command prompt run the following command from within the application's bin folder.
dns-updater.[env] [...options]
Replace [env] with the appropriate extension for your system.

  • Linux/MacOS Bash: .sh
  • Windows CMD: .cmd
  • Windows Powershell: .ps1
    For the powershell script to be able to run you must enable scripts in the powershell console which are disabled by default.

Also, replace [...options] by the desired running options (see below).

If you want you can also make shortcuts to the run the application (see example below).

Options

  • --usage

    Show a text based help message.

  • --help or -h

    Show this help page.

  • --once or -o

    Update all the configured DNS once.

  • --service or -s

    Run the application as a service.
    The update will happen at regular interval based on the serviceTimeout option in the configuration file.

  • --config or -c

    Specify the location of the configuration file.
    The default location is at the root of the application's folder.

  • --gui

    Run the graphical configurator.
    You can only use '--config' at the same time.

Examples

Update all configured DNS once
CMD: dns-updater.cmd --once

Run the graphical configurator with an irregular configuration file location.
Bash: dns-updater.sh --config ~/myconfig.json --gui

Run the application as a service.
Powershell: dns-updater.ps1 --service

Making a shortcut to run the gui on Windows.

  • Right click where you want the shortcut.
  • Click Browse... and select dns-updater.cmd in the application's bin directory.
  • Add the desired arguments, in our case: --gui
    You should have something similar to this in the input box: C:\dns-updater\bin\dns-updater.cmd --gui
  • Click Next and give the shortcut a name.
  • Finish!
